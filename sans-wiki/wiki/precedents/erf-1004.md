# ERF 1004 — Second Dwelling with Parking Shortfall

## Property Details
- **ERF Size:** 1200m²
- **Existing improvements:** Single dwelling (approved 2010), double garage (parking for 2)
- **Proposed works:** 45m² second dwelling (granny flat) to rear garden + new driveway

## Departments Reviewed
| Dept | Status | Notes |
|------|--------|-------|
| BC | ✅ APPROVED | All dimensions compliant |
| RSP | ⚠️ CONDITION | Parking relaxation motivation required |
| MH | ✅ APPROVED | |

## Key Decision Points

### Parking (Clause F1, SANS 10400 / Tshwane Scheme)
- Existing: 2 bays (double garage) ✅ for main dwelling
- Second dwelling requires 1 additional bay
- Existing site has space for 2 total — no room for third
- **Parking shortfall: 1 bay**

### Relaxation Route
- Tshwane Scheme Clause 27 allows relaxation of parking requirements where:
  - Site constraints prevent provision
  - Public transport within 500m
  - Engineer certifies shared use arrangement

Owner submitted:
1. Motivation letter citing constrained erf (large trees on one side, slopes on other)
2. Engineering letter confirming shared parking arrangement (garage bay used flexibly)
3. Google Maps screenshot showing bus stop 350m away

### Second Dwelling Size (Tshwane Scheme Clause 20)
- Proposed: 45m² — **within 80m² limit** ✅
- Compliant with second dwelling policy

### Coverage (Clause H2)
- Existing: 32%
- Second dwelling: +3.75%
- **Total: 35.75%** ✅ — well within 60%

## Decision
**CONDITIONAL APPROVAL** — granted on provision of:
1. Engineer's certificate confirming shared parking arrangement with diagram
2. Amended site plan showing the shared parking layout dimensioned

## Lessons Learned
- Parking relaxations are commonly granted where public transport exists
- Engineer letter specifying "flexible/shared parking" must include a sketch
- Second dwellings under 80m² are relatively straightforward — the parking was the only issue
- Always check Tshwane Scheme Clause 27 before rejecting on parking grounds
