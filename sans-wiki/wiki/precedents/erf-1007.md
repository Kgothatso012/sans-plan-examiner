# ERF 1007 — Dwelling Addition with Fire Compartment Size Correction

## Property Details
- **ERF Size:** 750m²
- **Existing improvements:** Single dwelling (approved 2018)
- **Proposed works:** 55m² double-storey addition (lounge downstairs, bedroom upstairs)

## Departments Reviewed
| Dept | Status | Notes |
|------|--------|-------|
| BC | ⚠️ CONDITION | Compartment size exceeded — corrected with firewall |
| RSP | ✅ APPROVED | |
| MH | ✅ APPROVED | |

## Key Decision Points

### Fire Compartment Size (Clause B2, SANS 10400)
- R1 single dwelling: maximum 300m² floor area without fire wall
- Existing dwelling: 185m²
- Proposed addition: 55m²
- **Combined: 240m²** — within 300m² ✅

Wait — recalculation revealed different picture:

- Ground floor existing: 95m²
- Ground floor addition: 55m²
- First floor existing: 90m²
- **Total combined habitable area: 240m²** ✅ within 300m²

> But — owner had previously submitted a carport/braai area (unapproved) that added 40m²
> **Actual total: 280m²** — still within 300m²

Final calculation including all structures with roofing:
- Main dwelling + addition: 240m²
- Unapproved braai room: 40m²
- **Total: 280m²** ✅ — within 300m² compartment limit

### What Was Wrong
Initial submission showed "no fire wall required." Review found:
1. Unapproved braai room not included in schedule
2. The braai room shares a wall with the main dwelling and has no fire separation
3. This created a potential fire compartment issue

### Resolution
- Owner had two options: (a) regularise braai room with fire wall, or (b) demolish braai room
- Owner opted to install a 120-min rated fire wall between braai room and dwelling
- Structural engineer certificate submitted showing fire wall construction

### Coverage (Clause H2)
- 280m² structure / 750m² erf = **37.3%** ✅ — well within 60%

## Decision
**CONDITIONAL APPROVAL** — granted on provision of:
1. 120-min fire wall construction certificate for braai room / dwelling separation
2. Amended site plan showing fire wall location and fire-rated construction detail
3. Building inspector site sign-off after construction

## Lessons Learned
- Always ask about unapproved structures — owner didn't mention the braai room
- Fire compartment calculations must include ALL roofed structures
- 300m² limit for single dwellings is per compartment, not per structure
- A good site inspection would have caught the braai room before submission
