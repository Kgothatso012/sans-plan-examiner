# ERF 1010 — Boundary Wall Height Relaxation Granted

## Property Details
- **ERF Size:** 1,100m²
- **Existing improvements:** Single dwelling (approved 2009), perimeter wall 1.8m high
- **Proposed works:** Increase front boundary wall height from 1.8m to 2.2m + automated sliding gate

## Departments Reviewed
| Dept | Status | Notes |
|------|--------|-------|
| BC | ✅ APPROVED | Structural adequacy confirmed |
| RSP | ✅ APPROVED | Height relaxation granted |
| MH | ✅ APPROVED | |

## Key Decision Points

### Boundary Wall Height (Clause H3, SANS 10400 / Tshwane Scheme)
- SANS 10400-H3: Maximum front boundary wall height: **1.8m**
- Tshwane Scheme Clause 20: Maximum front boundary wall: **1.8m**
- **Proposed: 2.2m** — exceeds by 400mm (22% over)

### Relaxation Process (Tshwane Scheme)
- Boundary wall heights above standard can be relaxed by the Mhla Mus (Municipal Manager or delegate)
- Applicant must demonstrate:
  1. Security motivation (crime prevention)
  2. Visual impact on streetscape acceptable
  3. No impact on neighbouring properties (light, outlook)
  4. Structural adequacy of proposed wall

### Applicant's Motivation
- Submitted crime prevention motivation citing:
  - Waterkloof area has experienced 4 break-in incidents on same street
  - SAPS confirmation of neighbourhood priority area
  - Neighbouring properties within 3 houses also have walls above 1.8m
- Neighbour notification: No objections received from 3 adjacent erf owners

### Design Mitigation
- Applicant agreed to:
  1. Use decorative open brick pattern (50% perforated) for top 400mm — maintains visual openness
  2. Face brick to match existing wall texture
  3. No security toppings (razor wire) on top — maintains aesthetic
  4. Automated gate set back 3m from wall line — maintains streetscape

### Structural
- New wall section: engineering certificate confirming foundation and masonry design for 2.2m height
- Uses engineering brick, not face brick only — higher compressive strength

### Coverage (Clause H2)
- Wall footprint (0.3m wide × 2.2m high) — negligible area impact
- **Coverage: Unchanged** ✅

## Decision
**APPROVED** — granted with conditions:
1. Top 400mm of wall to be constructed with 50% open brick (decorative pattern)
2. No razor wire, electrified fencing, or similar toppings
3. Gate automation to be installed by registered electrician with COC
4. Engineering certificate to be filed with approved plans

## Lessons Learned
- Boundary wall relaxations are available but require proper motivation
- Crime prevention is a legitimate motivation — SAPS documentation helps
- Neighbour consent (or at least no objection) significantly strengthens application
- Design mitigation measures (open brick, decorative pattern) make approvals easier
- Never put security toppings on relaxed walls — immediate reason for enforcement
- Mhla Mus sign-off required — not a building control only decision
