# Interpretation: Ceiling Height Measurement

## Clause Reference
[[clauses/A2]]

## Core Principle
2400mm for habitable rooms, 2100mm for non-habitable. Measurement is from finished floor to finished ceiling.

## Accepted Deviations
| Scenario | Height | Condition |
|----------|--------|-----------|
| Loft conversion bedroom | 2400mm | Min 50% of floor area ≥2400mm |
| Kitchen extension | 2400mm | Required - habitable |
| Garage conversion | 2100mm | Non-habitable, no sleeping |
| Bulkhead above stairs | 2100mm | If not in habitable room path |

## Rejected Examples
| Scenario | Rejected Because | Measurement |
|----------|------------------|-------------|
| Bedroom with 2200mm | Below minimum | Finished floor to ceiling |
| Kitchen 2100mm | Kitchen is habitable | Must be 2400mm |
| Measure to beam | Including structural | Measure to finished ceiling |

## Joe's Notes
- Ask for "finished floor to finished ceiling" on section drawing
- Accept: 2350mm if documented as existing building with SAJHR variance
- Flat roof with beam: measure to lowest point if <2400mm
- Granny flat: habitable = living/bedroom, non-habitable = bathroom/kitchen
