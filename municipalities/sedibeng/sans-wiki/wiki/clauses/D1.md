# Clause D1: Stair Width

## Requirement
Minimum width of stairs shall be 900mm for residential buildings.
Minimum width of stairs shall be 1000mm for commercial buildings.
Stairs serving more than 50 persons: minimum 1100mm.

## Measurable Criteria
- [ ] Residential: min 900mm clear width
- [ ] Commercial: min 1000mm clear width
- [ ] Measure between handrails or wall to handrail

## Common Violations
- Spiral stairs with < 800mm diameter
- Under-stair storage reducing width to < 900mm
- Mezzanine stairs in commercial with 850mm width
- Door swing encroaching into stair width

## Accepted Deviations
| Scenario | Allowed Width | Condition |
|----------|---------------|-----------|
| Single dwelling < 100m² | 850mm | No fire escape requirement |
| Existing building conversion | 850mm | Practical constraint documented |
| External fire escape | 850mm | Escape route only |

## COJ / Emfuleni Notes
- COJ enforces via Municipal Spatial Planning and Land Use Management By-Law 2016
- Duplexes: 900mm minimum (no COJ exceptions — same as national)
- Townhouses (Res 2): 850mm accepted if total floor area < 100m²
- **SANS 10400-D:** 900mm residential, 1000mm commercial. COJ enforces strictly.
