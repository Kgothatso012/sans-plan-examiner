# Interpretation: Site Coverage Calculation

## Clause Reference
[[clauses/H2]]

## Core Principle
Site coverage measures the percentage of the site covered by building footprint. Tshwane maximum is 60% for Residential 1, but includes all impervious surfaces.

## Accepted Calculations
| Item | Included | Notes |
|------|----------|-------|
| Main building | Yes | Floor area at ground level |
| Garage/carport | Yes | Attached or detached |
| Outbuilding | Yes | If >10m² |
| Covered patio | Yes | If enclosed on 2+ sides |
| Pool deck | No | Uncovered, permeable |
| Garden paths | No | Permeable |
| Driveway | Yes | Concrete/asphalt |

## Common Mistakes
- Not including garage in coverage
- Excluding covered walkways
- Forgetting projecting balconies on upper floors

## Joe's Notes
- Ground floor × 1 = coverage % (not total floor area)
- Upper floor balconies: if covered, count 50% projection
- Example: 500m² erf, 300m² ground + 150m² upper = 60% (300/500)
- Bonus: 40% must remain permeable for storm water
