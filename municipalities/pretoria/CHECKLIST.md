# TSHWANE BUILDING PLAN COMPLIANCE CHECKLIST

## SECTION A: ERF DETAILS
- [ ] Erf number identified
- [ ] Erf size (m²) recorded from plan or SG diagram
- [ ] Zoning confirmed (R1 / R2 / etc.) from zoning certificate
- [ ] Title deed restrictions reviewed (Annexure L conditions?)
- [ ] GDARFE (Reg 891) conditions noted
- [ ] Land use rights confirmed (primary use = dwelling house)
- [ ] Property owner details present
- [ ] Surveyor-general diagram reference present

## SECTION B: SITE DEVELOPMENT PLAN
- [ ] North arrow present
- [ ] Scale stated (e.g., 1:100, 1:200)
- [ ] Site coverage calculated ≤ 60% (R1)
- [ ] FAR calculated ≤ 0.3 (R1)
- [ ] Floor area ratio = Total floor area ÷ Erf area
- [ ] Building lines shown: 3m street, 1.5m/2.0m sides, 3m rear
- [ ] Adjacent street name(s) identified
- [ ] Existing structures on adjacent properties noted
- [ ] Site access / driveway position shown

## SECTION C: CALCULATIONS
- [ ] Site coverage = Building footprint ÷ Erf area ≤ 60%
- [ ] FAR = Total floor area ÷ Erf area ≤ 0.3
- [ ] Building height measured to wallplate ≤ 10m
- [ ] Building lines measured correctly (per plan scale)

## SANS 10400 CHECKLIST

### A — GENERAL
- [ ] NBR application: SANS 10400-A
- [ ] Building plan drawn to scale
- [ ] Professional architect/engineer stamp present

### B — SITE
- [ ] Drainage point indicated
- [ ] Sewer connection point shown
- [ ] erf boundary dimensions match SG diagram
- [ ] Slope / level datum stated

### C — DISSIPATION
- [ ] Soil-bearing capacity assumed (standard: 150 kPa unless geotech report)
- [ ] No mining/dolomitic risk indicated (verify)

### D — FOUNDATIONS
- [ ] Foundation type specified (strip / raft / pad)
- [ ] Foundation dimensions shown
- [ ] Damp-proof course (DPC) indicated

### G — EXCAVATION
- [ ] Adjacent structures protected during excavation
- [ ] Lateral support to neighbouring properties

### H — FLOORS
- [ ] Damp-proof membrane (DPM) indicated
- [ ] Screed thickness noted
- [ ] Floor levels relative to NBR datum

### J — WALLS
- [ ] Wall thickness specified (min 110mm for brick)
- [ ] Structural wall positions shown
- [ ] Openings (windows/doors) sized correctly

### K — ROOFS
- [ ] Roof covering material specified
- [ ] Roof insulation indicated (SANS 10400-TA)
- [ ] Pitch stated

### L — GLAZING
- [ ] Safety glazing identified (SANS 10400-L)
- [ ] Glass type specified for each window/door

### M — STAIRS
- [ ] Tread × Riser: min 250mm × max 200mm
- [ ] Going (tread depth) ≥ 250mm
- [ ] Headroom ≥ 2100mm
- [ ] Balustrade height ≥ 1000mm
- [ ] Handrail both sides
- [ ] Open riser not permitted ( dwellings)

### N — FIRE
- [ ] Fire resistance of walls ( Table of fire resistance ratings)
- [ ] Means of escape (2 exits if habitable rooms > 4.5m from exit)
- [ ] Fire extinguisher location noted
- [ ] Fire-rated door indicated (bedroom > 4.5m from front door)

### P — DRAINAGE
- [ ] Sewage disposal: municipal sewer connection
- [ ] OR: Septic tank / conservancy tank approved
- [ ] No discharge to storm water

### T — VENTILATION
- [ ] Ventilation opening ≥ 5% of floor area (SANS 10400-T)
- [ ] OR: Mechanical ventilation specified
- [ ] Kitchen extraction fan indicated

### TA — ENERGY
- [ ] Wall insulation R-value stated
- [ ] Roof insulation R-value stated
- [ ] Glazing U-value stated
- [ ] Water heating: solar/gas/heat pump specified

- [ ] Zoning confirmed as R1 (Residential 1)
