# Pretoria (Tshwane) — Plan Examiner

## Municipality
- **Name:** City of Tshwane Metropolitan Municipality
- **Region:** Gauteng Province
- **Website:** www.tshwane.gov.za
- **Scheme:** Tshwane Land Use Scheme, 2001 (Revised 2015 via GN 765 of 2015)

## Scheme Controls — Residential 1 (R1)

| Control | Value |
|---------|-------|
| Coverage | 60% max |
| Height | 10m max |
| FAR | 0.3 max |
| Street BL | 3.0m (erf < 1,000m²) |
| Side BL | 1.5m (single) / 2.0m (double) |
| Rear BL | 3.0m |
| Parking | 1 bay/dwelling |

## Key Clauses
- Clause 9–11: Building lines
- Clause 25: Floor Area Ratio (FAR)
- Clause 26: Height limits
- Clause 27: Site coverage
- Clause 28: Parking requirements
- Clause 44: Rezoning
- Clause 45: Departure
- Clause 46: Consent use

## Workflow
1. OCR the building plan → extract text
2. Run CHECKLIST.md against extracted text
3. Cross-reference with SCHEME-SUMMARY.md and sans-wiki clauses
4. Draft compliance report → approval / rejection with reasons

## Important
- Always check for **Annexure L** conditions on title deed — they override R1 controls
- Check GDARFE (Reg 891) for rate-related conditions
- Verify zoning certificate matches the application
- Pre-1970 buildings may have heritage status — check with Tshwane Heritage Section
