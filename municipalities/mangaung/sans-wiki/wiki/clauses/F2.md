# Clause F2: Driveway Width

## Requirement
Driveways shall provide adequate width for vehicle access and turning.

## Measurable Criteria
- [ ] Single driveway width: min 3.5m
- [ ] Double driveway width: min 5m
- [ ] Gradient max 1:6 (16.7%)
- [ ] Clear height: min 2.1m

## Common Violations
- Driveway too narrow (<3.5m single)
- Gradient too steep
- Clearance to boundary wall <1m
- Garage door width <2.7m
