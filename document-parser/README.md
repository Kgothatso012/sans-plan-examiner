# Tshwane Zoning FAQ Bot

Answers common zoning questions via Telegram.

## Setup

```bash
cd tshwane-zoning-faq-bot
npm install
# Set TELEGRAM_BOT_TOKEN in .env
npm start
```

## FAQ Topics

| Keyword | Answer |
|---------|--------|
| second dwelling | 80m² max, same erf |
| pool | 50m² max, 1.5m boundary |
| coverage | 60% residential |
| height | 2 storeys default |
| subdivision | 3-6 months |
| apply | Form B + plans |
| time | 2-12 weeks |
| cost | R500-R1500 |

## Deploy

Render/Railway - set `TELEGRAM_BOT_TOKEN`
